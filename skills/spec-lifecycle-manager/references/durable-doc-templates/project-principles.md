## Decision Questions

1. Does this work support the project purpose and mission?
2. Does this solve the stated problem better than existing project mechanisms?
3. Is this project the right owner, or should another project/tool own it?
4. Can the change be validated or governed in a way that fits this project?
5. Is the work worth doing now, or should it stay in backlog or roadmap?
